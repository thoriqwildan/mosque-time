# Setup Instructions

1. Clone the GitHub repository.

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder.

   ```bash
   cd jam-masjid-php
   ```

3. Create the data.json file.

   ```bash
   touch data.json
   ```

4. Set permissions for data.json.

   ```bash
   chmod 777 data.json
   ```

5. Set permissions for the project folder.

   ```bash
   chmod 777 .
   ```

6. Run the code.
   ```bash
   php -S 0.0.0.0:[port]
   ```
